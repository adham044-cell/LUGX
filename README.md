# LUGX Gaming Website

**LUGX** - موقع ويب احترافي لعرض ألعاب الفيديو

## الوصف (Description)
LUGX Gaming هو موقع ويب حديث وجذاب لعرض ألعاب الفيديو. تم تطويره بواسطة **ADHAM SAMIR** باستخدام HTML5 و CSS3.

## المميزات (Features)
✅ تصميم عصري وجذاب  
✅ أداء سريع وسلس  
✅ قابل للاستخدام على جميع الأجهزة  
✅ تخطيط منطقي وسهل التنقل  

## الأقسام الرئيسية
- **الصفحة الرئيسية**: عرض ترحيبي جذاب مع صورة بانر
- **الألعاب الرائجة**: عرض أحدث الألعاب الشهيرة
- **الألعاب الأكثر لعباً**: قائمة الألعاب الأكثر انتشاراً
- **الفئات الأساسية**: تصنيفات مختلفة للألعاب
- **تواصل معنا**: نموذج للتواصل المباشر

## معلومات التطوير
- **المطور**: ADHAM SAMIR
- **اللغات المستخدمة**: HTML5, CSS3
- **الترخيص**: مفتوح المصدر للاستخدام التجاري
